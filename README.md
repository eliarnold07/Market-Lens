MarketLens

MarketLens is a stock and portfolio analysis platform that I’ve been building to learn more about finance, analytics, AI, and software development. The project originally started as a simple stock dashboard, but over time I expanded it into a more intelligent portfolio analysis system focused on strategy alignment, risk analysis, and explainable recommendations.

The platform analyzes both individual stocks and full portfolios. It looks at things like concentration risk, volatility, diversification, sector exposure, momentum, and overall portfolio balance. One of the main goals of the project has been making the analysis easy to understand instead of just throwing charts and financial terms at the user.

A large part of the project has focused on building a recommendation system that explains why a portfolio does or does not fit a certain investing strategy. Instead of acting like an “AI stock picker,” the system tries to explain what is driving portfolio behavior and what types of changes could improve alignment with strategies like Conservative Investor, Balanced Investor, Aggressive Growth, and Momentum Trader.

A lot of recent development has gone into improving the intelligence layer of the platform, especially around recommendation prioritization, portfolio reasoning, readability, and reducing repetitive AI-generated commentary. The goal has been to make the analysis feel more like a real portfolio strategist and less like generic chatbot output.

The project was built using HTML, CSS, JavaScript, financial market APIs, the OpenAI API, and Git/GitHub for version control and iterative development.

Some future improvements I plan to continue working on include caching and performance optimization, portfolio persistence, backtesting validation, portfolio comparison tools, and eventually deployment/authentication features.

Disclaimer: MarketLens is an educational project and should not be considered financial advice.
